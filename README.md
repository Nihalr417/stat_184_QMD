# stat_184_QMD

# What this project is about?
This project analyzes calcium measurements over time and compares two treatment groups. The goal is to clean, reshape, and visualize the data to understand trends across years.

# Where the data come from?
The dataset contains calcium measurements collected across multiple years for different treatment groups. It was provided as part of a class assignment.

# Current plan for the project
- Clean and format the dataset
- Convert data into long format
- Create visualizations (line plots)
-Interpret trends between treatment groups
